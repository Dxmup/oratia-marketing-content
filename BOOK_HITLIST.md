# Book Hitlist

Purpose: a practical source-of-truth list for the Content Factory.

Selection rules:
- books should be at least moderately popular or respected in sales, negotiation, persuasion, buying psychology, positioning, messaging, or decision-making
- psychology books are allowed when they can be translated into sales, trust-building, owner acquisition, objection handling, or operator behavior
- priority is not literary prestige; priority is reliable content generation that VRM operators will actually want to read
- this file is the canonical queue for `next unprocessed book` decisions until replaced by a better structured tracker

Status key:
- processed = content exists on disk and is verified
- queued = approved for future processing
- maybe = potentially useful but lower priority or more abstract

Priority key:
- P1 = highest-priority next candidates
- P2 = strong secondary candidates
- P3 = useful but more optional/abstract
